## 🐝 Proyectos Destacados — Open Source

### [Hive](https://github.com/johpaz/hive) — Gateway de Agentes IA 🐝⭐
> **44 Stars** · **12 Forks** · **232 Commits** · **51K+ líneas de código**

Gateway de IA orquestado — un enjambre de agentes especializados que trabajan juntos bajo la coordinación de un gateway central. Local-first, multi-canal, open source. Construido desde Colombia para el mundo.

| Característica | Detalle |
|----------------|---------|
| **Runtime** | Bun + SQLite |
| **Arquitectura** | Native Agent Loop (sin LangChain) |
| **Tools** | 73 nativas (navegador, filesystem, cron, Office) |
| **Skills** | 30 integradas |
| **Canales** | WebChat, Telegram, Discord |
| **Models** | Gemini, Anthropic, OpenAI, Groq, Ollama |
| **Compatibilidad** | Raspberry Pi, VPS, Docker, binario standalone |

[⭐ Ver en GitHub](https://github.com/johpaz/hive) · [🌐 hiveagents.io](https://hiveagents.io) · [💬 Discord](https://discord.gg/hive)
